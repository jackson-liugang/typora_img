## 什么是JDK、JRE

**JDK(Java Development kit   JAVA开发工具包)** 包括： JRE +Java开发工具（javac.exe,java.exe,javadoc.exe）



**JRE（Java Runtime Environment  Java运行环境）**包括：Java虚拟机（JVM Java Virtual Machine）+ Java程序所需的核心类库
